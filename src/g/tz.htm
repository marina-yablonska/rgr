ТЕХНІЧНЕ ЗАВДАННЯ

на виконання розрахунково-графічної роботи з дисципліни «Моделювання»

         Тема: Моделювання роботи кафе на пляжі, Варіант «BeachCafee».

         Виконавці: Артюх О.О., Гриць А.В., Яблонська М.І. гр.КІ-141.

         Опис системи

        На пляжі є кафе. Офіціанти перевіряють, чи є нові відвідувачі. Якщо є, офіціант обслуговує клієнта (приймає замовлення), після цього іде до кухні і замовляє страву. На кухні кухар готує страву і повідомляє про виконане замовлення. Одержавши готову страву, несе її відпочиваючому. Якщо страва готувалась більше ніж вибраний період часу, відвідувач покидає кафе.  Інакше, офіціант приносить замовлення клієнту і розраховується. 

         Завдання на проектування

         Створити Java застосування для імітаційного моделювання кафе на пляжі, яке дозволить:

         - налаштування параметрів моделі, а саме: інтервал між появою нових відвідувачів, кількість клієнтів, розмір черги замовлень на кухні, кількість готових та виданих страв, кількість відвідувачів, що не дочекались замовлення, площадки для тестування, час моделювання, закони розподілення для випадкових величин часу приготування та споживання страви.

         - проведення тестових запусків моделі при різних налаштуваннях з динамічною індикацією черги відвідувачів, довжини черги замовлень на кухні, кількості вільних офіціантів, та можливості виведення протоколу роботи моделі під час тестових запусків;

         - проведення експериментів для отримання статистичних характеристик для довжини черг та часу простою офіціантів та відвідувачів.

         - отримання залежностей для середніх значень вказаних вище параметрів від кількості офіціантів;

         - отримання перехідних процесів для середніх значень довжини кожної з черг.
